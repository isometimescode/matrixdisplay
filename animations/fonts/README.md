BDF bitmap fonts, sourced from the
[`fonts/`](https://github.com/hzeller/rpi-rgb-led-matrix/tree/master/fonts)
directory of `rpi-rgb-led-matrix` (the same fonts the real hardware library
ships with).

- `7x13.bdf` — public domain.
- `5x7.bdf`, `6x9.bdf` — public domain. More compact than `7x13`, tried as
  a middle ground after `tom-thumb.bdf` (3px wide) turned out too small to
  tell characters like M/N apart.
